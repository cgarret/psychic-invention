## Organization Pattern

The current home note combines daily-note style content with a knowledge-garden approach.
The vault appears to use both MOC navigation and topic folders rather than a single strict book structure.

## Notable Conventions

- Top-level folders separate knowledge domains and workflows.
- MOC-style files connect larger views of the vault.
- Some content is organized around daily notes and active growth tracking.
- Atlas files act as the main entry points for higher-level navigation.
