# System Naming

These prefixes organize everything alphabetically and in your mind:
00_NOT   = Note, idee, frammenti

01_SCN   = Scene  
02_CH    = Capitoli  
03_ARC   = Archi narrativi  
04_WB    = Worldbuilding  
05_GEN   = Genealogie  
06_MAP   = Mappe (diagrammi, timeline)  
07_RES   = Ricerca, fonti  
08_DIA   = Diagrammi modulari  
09_FIN   = Versioni finali  
10_MAC   = Macro  

Esempi:
00_NOT_AppuntoSparso_2026-04-09.md
01_SCN_IngressoBosco_2026-04-09_r2.md
05_GEN_FamigliaMarras_2026-04-09_r1.json
09_FIN_Capitolo03_2026-04-09.md

Grammatica del nome
[Livello]_[Modulo]_[TitoloBreve]_[Data]_[Rev]
